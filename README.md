# Yaphet2015 Tap

## How do I install these formulae?

```sh
brew tap Yaphet2015/tap
brew install ya-skills
brew install --cask prokcy
```

Or install directly:

```sh
brew install Yaphet2015/tap/ya-skills
brew install --cask Yaphet2015/tap/prokcy
```

## Formulae

- `ya-skills`: personal skill repository and `yk` CLI.

## Casks

- `prokcy`: desktop proxy client based on Whistle.

Upgrade Prokcy with:

```sh
brew upgrade --cask prokcy
```
